# snake-asm
a simple boot sector snake game in assembly 8086<br>



## Prerequisites
- nasm
- qemu
## how to run
to compile:

    make


to run:

    make run
##  debuuging
dubbgging qemu is not as straightforward as using gdb. firts, type

    make gdb

and than open a second terminal and type

    make gdb-run
